## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/genesis-enewsletter.git
   cd genesis-enewsletter
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in your browser:**
   ```
   http://localhost:5173
   ```
